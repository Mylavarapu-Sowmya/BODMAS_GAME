# 🎮 BODMAS Bus Jam Game

An interactive web-based game designed to teach the **BODMAS (Order of Operations)** concept through a fun, visual, and engaging gameplay experience.

## 🧠 Concept

This game helps users understand the **order of operations (BODMAS)** by allowing them to:

- Interact with expressions visually
- Select the correct operator based on priority
- Solve expressions step-by-step
- Learn through instant feedback

---

## 🎯 Features

- 🎮 Interactive game interface
- 🧩 Visual expression blocks (numbers & operators)
- ✅ Real-time feedback (Correct / Wrong)
- 🔄 Step-by-step solving logic
- ⭐ Score tracking system
- 🎯 Level progression
- 🎨 Modern UI with animations
- 📱 Mobile responsive design

---

- **HTML** – Structure

---

## 🎲 How to Play

1. A mathematical expression is displayed.
2. Click the operator that should be solved first according to BODMAS.
3. 
   - ✅ Correct → Expression updates step-by-step  
   - ❌ Wrong → Feedback is shown  
4. Continue until the expression is fully solved.
5. Progress through levels and improve your score.
